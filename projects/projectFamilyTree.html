Software de Árbol Genealógico

En este proyecto universitario, desempeñé el rol de desarrollador full stack y lideré el desarrollo de un emocionante Software de Árbol Genealógico. Utilizando Java y JavaFX para el frontend, y CSS para mejorar el diseño y la presentación visual, implementamos una solución que permitía a los usuarios crear y visualizar árboles genealógicos de manera intuitiva. Además, optamos por utilizar serialización JSON como base de datos para almacenar la información de las relaciones familiares.

Tecnologías utilizadas:

  • Lenguaje de Programación: Java
  • Framework: JavaFX
  • Diseño: CSS
  • Base de Datos: Serialización JSON

El resultado de este proyecto universitario fue un Software de Árbol Genealógico funcional y práctico, que proporcionó una forma innovadora y eficiente de crear, visualizar y explorar las relaciones familiares. Durante el desarrollo del proyecto, adquirí habilidades sólidas en el desarrollo con Java y JavaFX, así como en la implementación de bases de datos con serialización JSON. Esta experiencia demuestra mi capacidad para diseñar y desarrollar soluciones creativas y educativas que utilizan la tecnología para facilitar la comprensión y la exploración de las relaciones familiares en un árbol genealógico.
