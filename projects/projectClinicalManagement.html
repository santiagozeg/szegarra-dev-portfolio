Proyecto Gestión de Clínicas Médicas

En este proyecto universitario, desempeñé el rol de desarrollador full stack de un completo Sistema de Gestión de Clínicas Médicas. El objetivo principal fue proporcionar a las clínicas médicas una herramienta eficiente y segura para gestionar sus operaciones diarias, manteniendo el control de historiales clínicos de pacientes, programando citas médicas y administrando la información de médicos y pacientes.

Tecnologías utilizadas:

  • Backend: Dotnet (C#)
  • Frontend: Angular
  • Base de Datos: MySQL
  • Contenedores: Docker

El resultado de este proyecto fue un Sistema de Gestión de Clínicas Médicas funcional y sólido que permitía a las clínicas mejorar sus operaciones y brindar una mejor experiencia de atención médica a sus pacientes. Durante el desarrollo del proyecto, adquirí una valiosa experiencia en el uso de tecnologías como Dotnet, Angular, MySQL y Docker, y fortalecí mis habilidades como desarrollador full stack. Este proyecto universitario representa mi capacidad para enfrentar desafíos complejos y aplicar soluciones innovadoras en el campo del desarrollo de software.
