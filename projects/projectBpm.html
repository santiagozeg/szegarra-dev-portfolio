Software de Gestión de Flujos de Trabajo (Software BPM)

Durante mi pasantía en la empresa boliviana Jalasoft, asumí el rol de desarrollador full stack en un emocionante proyecto centrado en la creación y gestión de flujos de trabajo (Software Business Process Management - BPM). Mi principal objetivo fue desarrollar una plataforma que permitiera a los usuarios crear, visualizar y ejecutar flujos de trabajo de manera intuitiva y eficiente. Para lograr esto, trabajé tanto en el backend utilizando Dotnet, como en el frontend con ReactJS, además de integrar tecnologías como MySQL y Docker para asegurar un despliegue fácil y escalable.

Tecnologías utilizadas:

  • Backend: Dotnet (C#)
  • Frontend: ReactJS
  • Base de Datos: MySQL
  • Contenedores: Docker

El resultado de este proyecto de pasantía fue una Plataforma de Gestión de Flujos de Trabajo completa y efectiva, que proporcionó a las empresas una herramienta poderosa para mejorar la eficiencia de sus procesos internos. Durante este proceso, adquirí habilidades valiosas en el desarrollo con Dotnet y ReactJS, así como en la implementación de soluciones de bases de datos con MySQL y la utilización de Docker para asegurar una entrega sin complicaciones. Este proyecto demuestra mi capacidad para abordar desafíos complejos y crear soluciones innovadoras en el ámbito del desarrollo de software empresarial.
