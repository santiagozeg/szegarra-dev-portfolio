Software de Gestión Educativa y Videoconferencias

Durante mi participación en este proyecto universitario, asumí el rol de desarrollador full stack y lideré la creación de una completa Plataforma de Gestión Educativa y Videoconferencias. Esta plataforma se asemejaba a aplicaciones conocidas como Zoom o Teams, y su objetivo principal fue proporcionar a estudiantes y educadores un entorno virtual donde pudieran interactuar y colaborar eficientemente. Para lograrlo, utilicé Node.js en el backend, y tecnologías web como CSS, HTML y JavaScript para el frontend. Además, implementé MongoDB para la base de datos y Docker para garantizar un despliegue fácil y rápido.

Tecnologías utilizadas:

  • Backend: Node.js
  • Frontend: CSS, HTML, JavaScript
  • Base de Datos: MongoDB
  • Contenedores: Docker

El resultado de este proyecto universitario fue una Plataforma de Gestión Educativa y Videoconferencias completa y funcional, que proporcionó un espacio virtual dinámico y eficiente para el aprendizaje y la colaboración. Durante el desarrollo de este proyecto, fortalecí mis habilidades como desarrollador full stack y adquirí experiencia en el uso de Node.js, MongoDB y tecnologías web. Esta experiencia demuestra mi capacidad para crear soluciones innovadoras que aborden necesidades educativas en un mundo cada vez más digitalizado.
